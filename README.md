## Usage

    Date.today.united_states_thanksgiving?
    Date.today.shopping_black_friday?

## Installation

Add this line to your application's Gemfile:

    gem 'a_special_day'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install a_special_day
